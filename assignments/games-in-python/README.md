# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build the classic word-guessing game using Python strings, loops, and user input. You'll create an interactive game where players guess letters to reveal a hidden word before running out of attempts.

## 📝 Tasks

### 🛠️ Build Hangman Game

#### Description

Create a Hangman game that implements core gameplay mechanics. The game should randomly select a word from a predefined list and allow players to guess letters. Display the current progress of guessing the word, track remaining attempts, and determine when the game ends with a win or loss condition.

#### Requirements

Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and show current progress (using underscore format like _ _ _)
- Track incorrect guesses remaining and display the count
- End when the word is guessed correctly or attempts are exhausted
- Display clear win/lose messages at the end of the game
- Use string manipulation, loops, and conditionals in the implementation
